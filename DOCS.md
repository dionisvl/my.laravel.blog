# Documentation

### Swagger Open Api

#### install

- `composer require "darkaonline/l5-swagger"`
- Run `php artisan vendor:publish --provider "L5Swagger\L5SwaggerServiceProvider"`    
  to publish new swagger-ui view and configuration
- `php artisan l5-swagger:generate`
